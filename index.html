Rating System
RD
Rating Deviation
RD measures how confident the system is in a player's rating. A low RD means the rating is reliable; a high RD means more uncertainty. New players start at RD 120. After enough games, RD can drop below 100 — only then does a player appear on the leaderboard.
σ
Volatility (σ)
Volatility measures how erratic a player's performance is. A higher σ means results are unpredictable — the system reacts more strongly to upsets. It's updated each match using the full Glicko-2 Illinois algorithm. Starting σ: 0.06. Capped at 0.30.
⏱ RD
RD Decay (Inactivity)
If you stop playing, your RD slowly increases using your personal volatility: φ* = √(φ² + σ²). This means volatile players decay faster than consistent ones. RD is capped at 200. If RD rises above 100 you'll be hidden from the leaderboard.
G2
Glicko-2 Algorithm
Full Glicko-2 implementation including per-player volatility updates via the Illinois root-finding method. Every match updates rating, RD, and volatility based on expected outcome — upsets gain more, expected wins gain less. Starting rating: 1500.
